# Исторические даты

[Посмотреть на проект](https://alexandrger.github.io/historical-dates/).

## Инструкция по установке

1. Клонировать репозиторий

```
git clone https://github.com/AlexandrGer/historical-dates.git
```

2. Установить зависимости

```
npm install
```

3. Запускаем проект

```
npm run start
```
